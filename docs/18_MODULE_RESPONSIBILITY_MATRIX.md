3. نموذج RASCI المعتمد
الرمز	الدور	الوصف
R	Responsible (مسؤول)	المنفذ الفعلي للعمل والصيانة
A	Accountable (معتمد)	المالك النهائي الذي يعتمد القرارات والتسليمات
S	Supportive (داعم)	يقدم الدعم الفني عند الحاجة
C	Consulted (مستشار)	تُطلب استشارته قبل التنفيذ (اتصال ثنائي)
I	Informed (مُخطر)	يُبلغ بعد التنفيذ (اتصال أحادي)

    │
    ▼
المهندس المسؤول (R)
    │ لا يستطيع الحل
    ▼
مراد (كبير المهندسين المعماريين)
    │ خلاف مع مهندس آخر
    ▼

    │ لا يوجد إجماع
    ▼

    ▼
