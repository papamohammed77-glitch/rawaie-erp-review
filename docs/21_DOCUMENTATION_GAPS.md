#	الفجوة	الوصف	التأثير	المساهم
1	سياسات RLS الكاملة	لا يوجد توثيق لنص SQL الكامل لسياسات RLS على كل جدول. وثيقة 12_SECURITY_MODEL.md تذكر أمثلة فقط.	لا يمكن لمدقق أمني التحقق من أمان البيانات. لا يمكن لمهندس جديد إعادة إنشاء السياسات.	مراد
2	دليل النشر الكامل (Deployment Guide)	لا يوجد توثيق لعملية النشر على Cloudflare Pages خطوة بخطوة: إعداد المشروع، ربط المستودع، تكوين النطاقات، إعدادات البيئة.	لا يمكن لـ DevOps جديد نشر النظام.	مراد، حامد
3	توثيق RPC Functions	دوال PostgreSQL مثل get_trial_balance و get_profit_loss و get_budget_vs_actual غير موثقة: لا نعرف هل هي منشأة فعلاً، ولا معلماتها الدقيقة.	التقارير المالية قد لا تعمل (تستدعي RPC غير موجودة).	مراد
4	ملف DATABASE_SCHEMA.sql	لا يوجد ملف SQL كامل لإنشاء جميع الجداول الـ 51 دفعة واحدة.	لا يمكن تهيئة قاعدة بيانات جديدة.	حامد
5	ملف EDGE_FUNCTIONS_DEPLOYMENT.md	لا يوجد دليل تفصيلي لنشر كل Edge Function على حدة (أوامر Supabase CLI).	لا يمكن نشر الدوال على بيئة جديدة.	حامد
6	ملف ENVIRONMENT_VARIABLES.md	لا يوجد .env.example أو قائمة كاملة بجميع متغيرات البيئة (Supabase، Edge Functions، Cloudflare).	فشل النشر على بيئة جديدة.	حامد، محمود
7	Entity Relationship Diagram (ERD) كامل	ملف DATABASE_DOCUMENTATION.md يحتوي على ERD مبسط، لكنه لا يشمل جميع الجداول والعلاقات (51 جدولاً).	صعوبة فهم العلاقات بين الجداول.	محمود
